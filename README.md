# Chroma Vector Database Java Client

This is an implementation of the Chroma Vector Database API for Java `17+`.

This client works with Chroma API V2

## Features

- Full supported features are listed at [here](./FEATURES.md)

## Installation

Add Maven dependency:

```xml

<dependency>
    <groupId>io.github.locxngo</groupId>
    <artifactId>chroma-client</artifactId>
    <version>1.0.1</version>
</dependency>
```
